Create a program that will convert seconds to a readable format. 
5212 seconds should be converted to 1 hour, 26 minutes, and 52 seconds.

Create a function that will convert seconds to hours and return 
the value of an object with two keys: “hours” and “remainder".


1. Define a function named “toHours” with one parameter named “seconds.”


function toHours(seconds) {
  // code here
  }


2. Create a variable named “hours.” Set its value to Math.floor(seconds / 3600);


function toHours(seconds) {
    let hours = Math.floor(seconds / 3600);
  }


3. Create another variable named “remainder” and set its value to a statement 
that computes the remainder of “seconds” divided by 3600.
Hint: To compute the remainder, use the modulus operator (%)


function toHours(seconds) {
    let hours = Math.floor(seconds / 3600);
    let remainder = seconds % 3600; 
  }


4. Use the “return” keyword to set the function output as an object with two key
value pairs. The first key is “hour” set to the “hour” variable. The second key is 
“remainder” set to the “remainder” variable.
Hint: The returned value should be of the object data type. Here’s an example 
how it can look: return {field1: variable_value1, field2: variable_value1}.


function toHours(seconds) {
    let hours = Math.floor(seconds / 3600);
    let remainder = seconds % 3600;  
    return { hour: hours, remainder: remainder, };
}
